# 居酒屋オーダー・売上管理システム

Google Apps Script (GAS) で動作する、スマホ／タブレット対応の居酒屋向けオーダー・会計システム。

## 構成

| ファイル        | 役割                                    |
| --------------- | --------------------------------------- |
| `code.gs`       | サーバー処理（席管理・注文・会計・集計） |
| `index.html`    | UI（Bootstrap 5、モバイル最適化）        |

## 主な機能

- カウンター（1〜8番）／テーブル（A卓・B卓）の席管理
- お通し代の自動計上（席を開く時に人数入力）
- カウンター席のグループ化／解除
- 各席にリアルタイム合計金額を表示
- 「今日／今週（月曜開始）／今月」の売上サマリ
- 会計時に `SalesHistory` シートへ転記、`ActiveOrders` から削除
- 税率10％固定

## 使用シート（自動生成）

| シート名       | 列                                              |
| -------------- | ----------------------------------------------- |
| `ActiveOrders` | 席番号 / 商品名 / 単価 / 数量 / 小計 / タイムスタンプ |
| `SalesHistory` | 日時 / 席番号 / 商品名 / 単価 / 数量 / 税込合計 |
| `Menu`         | 商品名 / 単価 / カテゴリ                        |
| `SeatState`    | 席番号 / グループリーダー / 人数 / 開店時刻      |

## デプロイ手順

1. Google スプレッドシートを新規作成
2. **拡張機能 → Apps Script** を開く
3. デフォルトの `コード.gs` を `code.gs` の中身で置き換え
4. ファイル追加で `index` を作成（HTMLファイル）し、`index.html` の中身を貼り付け
5. **デプロイ → 新しいデプロイ → ウェブアプリ**
   - 実行ユーザー: 自分
   - アクセス: 任意（自分のみ／全員 など）
6. 発行された URL をスマホ／タブレットで開く

初回起動時、必要な4シートが自動生成され、サンプルメニュー（お通し含む）が投入されます。
